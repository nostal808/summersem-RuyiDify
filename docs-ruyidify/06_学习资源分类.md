## 6.3 推荐学习顺序（给新手/二开者）

1. **先理论**：`Dify知识库专题笔记.md` → `知识库01_切分索引与检索链路.md`
2. **建库实操**：`examples/21` → `01上传文档` → `03查看分段`（理解切分）
3. **检索调参**：`examples/04检索` → `16/17/18/19/20`（语义/混合/重排/阈值/TopK）→ 配 `知识库02`
4. **元数据**：`examples/12/13/14` → 配 `知识库03`
5. **应用闭环**：`examples/32` 用应用 API 做「检索+生成」问答
6. **进阶**：`知识库04_外部知识库API.md`（自研 RAG 接入）
> 顺序设计原则：先懂链路 → 再建库 → 再调检索 → 再上应用 → 最后外部集成。
